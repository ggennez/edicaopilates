
# Edição Pilates

Projeto frontend em React + Vite.

## Rodar localmente

Pré-requisito: Node.js 20+

1. Instale dependências:
   `npm install`
2. Rode em desenvolvimento:
   `npm run dev`
3. Build de produção:
   `npm run build`

## Deploy na Vercel

1. Suba este projeto no GitHub.
2. Na Vercel, clique em `Add New Project` e selecione o repositório.
3. A Vercel vai detectar Vite automaticamente.
4. Confirme as configurações:
   `Build Command`: `npm run build`
   `Output Directory`: `dist`
5. Clique em `Deploy`.

Este repositório já inclui `vercel.json` com essa configuração.
